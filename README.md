# CloudSpend – Serverless Expense Management System
A production-ready serverless expense tracking application built on AWS using API Gateway, Lambda, DynamoDB, and Cognito (coming soon). Designed to demonstrate scalable backend architecture, secure data handling, and real-world SaaS functionality.

Features
Secure REST API (CRUD operations)
Serverless architecture (no servers)
Real-time expense tracking
Category-based organization
Scalable DynamoDB storage
Cloud-native deployment

Architecture
Frontend: (coming soon – React)
API: API Gateway
Compute: AWS Lambda
Database: DynamoDB
Auth: (planned Cognito)

API Endpoints
POST   /expenses
GET    /expenses
PUT    /expenses/{expenseId}
DELETE /expenses/{expenseId}

What this demonstrates:
Serverless architecture design
API development
AWS service integration
Cloud security principles
Scalable backend systems
